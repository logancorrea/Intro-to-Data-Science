Algorithms, Geometry, and Optimization The classic paradigms in algorithm design include divide and conquer, greedy selection, dynamic programming, etc. But over the last 3-4 decades, a much richer set of techniques have transformed algorithm design. These include geometric and continuous "relaxations" for problems, leveraging tools from probability theory and optimization (e.g., gradient descent), and so on. The goal of this course will be to introduce some of these methods, and how they have had significant impact on the development of areas like data science.
